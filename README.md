# 国富実態算定計算機

TANAAKK流の国富の実態を算定するための計算機。政府がトラッキングすべき国のインジケータを、企業の連結BS・PL・CFと同様に、国民の所有する純資産・総資産として認識する。

## 考え方

- **連結BS視点**: 政府・企業・家計を一つの連結決算体として捉える
- **実態指標**: マネーサプライなど国によって定義が異なる数字に惑わされない
- **政府債務の評価**: GDP比ではなく国民純資産比で健全性を測る
